# AutoDRIVE Smart City Manager

## Install
Flask
```bash
$ pip install flask
```
Flask
```bash
$ pip install sqlite3
```

## Setup
Unix Bash (Linux, Mac, etc.):
    export FLASK_APP=scm_app
    export FLASK_ENV=development

Windows CMD:
    set FLASK_APP=scm_app
    set FLASK_ENV=development

## Run
cd <path/to/autodrive_scm/>
flask run
